# MR3001B.DroneScript.EqRosalia
Simulación en MATLAB del control de trayectoria de un dron de vigilancia autónomo. El proyecto implementa navegación por waypoints, modelado de perturbaciones ambientales y manejo de la incertidumbre estocástica en sensores de posicionamiento, evaluando el desempeño del lazo de control mediante métricas de error cuadrático medio.
